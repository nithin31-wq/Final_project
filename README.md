# Explainable Hybrid Ensemble Framework for Unsupervised Anomaly Detection

This project presents a hybrid anomaly detection system for industrial IoT sensor data using a CNN–LSTM Autoencoder and Isolation Forest. It enables accurate detection of anomalies without labeled data while providing sensor-level explainability.

## Features

* Unsupervised anomaly detection
* CNN–LSTM Autoencoder for temporal-spatial learning
* Isolation Forest for efficient anomaly scoring
* Hybrid ensemble model for improved performance
* Explainable AI for sensor-level insights

## Tech Stack

* Python
* PyTorch / TensorFlow
* NumPy, Pandas
* Matplotlib / Seaborn

## Usage

```bash
pip install -r requirements.txt
python src/main.py
```
